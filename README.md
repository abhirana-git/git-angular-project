# RecipeSharingPlatform
To run the project follow the below steps:
=>Run the below commands to run project
i.) npm install
ii.) Navigate to src folder to access db.json file
cd src
iii.) Run the json server command
npx json-server db.json
iv.) Now run the project 
npm start

Note: If project doesn't run try navigating one step back from src folder and then try again running the npm start command.
